# Focus Mode+ для Obsidian


**Режим глубокой концентрации с таймером, статистикой, уровнями и достижениями.**

Плагин превращает Obsidian в среду для глубокой работы (deep work), убирая визуальный шум и отвлекающие уведомления, при этом отслеживая прогресс через игровую механику.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎯 **Режим фокуса** | Ограничение интерфейса, отключение уведомлений |
| ⏱️ **Визуальный таймер** | Обратный отсчёт в статус-баре |
| 🔕 **Блокировка уведомлений** | Полная тишина во время сессии |
| 📊 **Статистика** | Часы, сессии, серии дней |
| 🏆 **5 уровней** | От Начинающего до Легенды |
| 🎖️ **7 достижений** | Мотивация и геймификация |
| 💡 **Рекомендации** | Персональные советы на основе паттернов |
| ⚙️ **Настройки** | Гибкая кастомизация |

## 📥 Установка

**Шаг 1. Откройте папку с хранилищем Obsidian**

1. Откройте папку хранилища Obsidian
2. Перейдите в `.obsidian/plugins/`
3. Скопируйте папку `focus-mode-plus` в `plugins/`
4. Выключите безопасный режим
5. Перезагрузите Obsidian 
6. `Настройки` → `Сторонние плагины` → Включить `Focus Mode+`

## Готово!

На левой панели появятся кнопки:
- **🎯 Фокус** — включить/выключить
- **📊 Статистика** — прогресс
- **💡 Советы** — рекомендации
